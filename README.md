# AgentOps Platform

Production-ready AI Agent Platform focused on:

- Agent Development
- CI/CD
- Agent Evaluation
- Docker
- GitHub Actions
- Monitoring
- Observability

## Docker Compose

The project uses Docker Compose to orchestrate the local development environment.

Current responsibilities:

- Build the API image
- Start the API service
- Load environment variables from `.env`
- Expose port 8000
- Automatically restart the service if it stops unexpectedly
